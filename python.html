Welcome To My Investment and Programming Showcase

Python est un langage de programmation non orientée objet qui est très polyvalent et facile à lire/écrire. Elle est principalement appliquée dans les systèmes de manipulation des données et intelligence artificielle à cause de sa vaste gamme de bibliothèques externes. Python a le plus de bibliothèques de C++ et Java, ce qui mène à plusieurs cas d'utilisation. De plus, c'est la langue la plus pour les débutants avec une courbe d'apprentissage très basse. En outre python est un langue interpreter et non compiler traditionnellement comme C++ et Java, ce qui permet le syntax d'être plus facile, cependant ça ralentit le vitesse du programme beaucoup comparer aux autres. Pour les pluparts de gens qui aspirait d'être programmeur c’est fort probable que le premier langage de programmation ils apprendrais est Python, cependant ca peut te mener très loin dans le chemin dans ton carriere tell que les applications dans les technologies émergents et innovative comme l’apprentissage automatique. Si tu veux être même un des autres types de programmeurs Python c’est un bon idee d’apprendre à cause de ces grandes bibliothèques et possibilités d’usages.
