Recherches et Apprentissages: Python

Que-ce-que c'est?

Python est un langage de programmation non orientée objet qui est très polyvalent et facile à lire/écrire. Elle est principalement appliquée dans les systèmes de manipulation des données et intelligence artificielle à cause de sa vaste gamme de bibliothèques externes. Python peut te mener très loin dans le chemin de ta carrière, notamment dans les applications des technologies émergentes et innovantes comme l'apprentissage automatique. Si tu veux être même un des autres types de programmeurs, Python est une bonne idée d'apprendre à cause de ses grandes bibliothèques et possibilités d'usages.

Avantages

Python a le plus de bibliothèques de C++ et Java, ce qui mène à plusieurs cas d'utilisation. De plus, c'est la langue la plus pour les débutants avec une courbe d'apprentissage très basse. En outre, Python est un langage interprété et non compilé traditionnellement comme C++ et Java, ce qui permet au syntaxe d'être plus facile. Pour la plupart des gens qui aspirent à être programmeurs, il est fort probable que le premier langage de programmation qu'ils apprendront est Python.

Desavantages

Cependant, cela ralentit la vitesse du programme beaucoup par rapport aux autres. De plus, Python n'est pas un bon option pour le développement des applications mobiles, et il utilise beaucoup plus de mémoire qu'un langue comme C++.

import random

print("Bienvenue à Connect Four, vous jouerez contre un ordinateur aléatoire.")
print("----------------------------------------------------------------------")

possibleLetters = ["A","B","C","D","E","F","G"]
gameBoard = [["","","","","","",""], ["","","","","","",""], ["","","","","","",""], ["","","","","","",""], ["","","","","","",""], ["","","","","","",""]]

rows = 6
cols = 7

def printGameBoard():
  print("\n     A    B    C    D    E    F    G  ", end="")
  for x in range(rows):
    print("\n   +----+----+----+----+----+----+----+")
    print(x, " |", end="")
    for y in range(cols):
      if(gameBoard[x][y] == "🔵"):
        print("",gameBoard[x][y], end=" |")
      elif(gameBoard[x][y] == "🔴"):
        print("", gameBoard[x][y], end=" |")
      else:
        print(" ", gameBoard[x][y], end="  |")
  print("\n   +----+----+----+----+----+----+----+")

def modifyArray(spacePicked, turn):
  gameBoard[spacePicked[0]][spacePicked[1]] = turn

def checkForWinner(chip):
  ### Check horizontal spaces
  for y in range(rows):
    for x in range(cols - 3):
      if gameBoard[x][y] == chip and gameBoard[x+1][y] == chip and gameBoard[x+2][y] == chip and gameBoard[x+3][y] == chip:
        print("\nJeu terminé", chip, "gagne ! Merci d'avoir joué :)")
        return True

  ### Check vertical spaces
  for x in range(rows):
    for y in range(cols - 3):
      if gameBoard[x][y] == chip and gameBoard[x][y+1] == chip and gameBoard[x][y+2] == chip and gameBoard[x][y+3] == chip:
        print("\nJeu terminé", chip, "gagne ! Merci d'avoir joué :)")
        return True

  ### Check upper right to bottom left diagonal spaces
  for x in range(rows - 3):
    for y in range(3, cols):
      if gameBoard[x][y] == chip and gameBoard[x+1][y-1] == chip and gameBoard[x+2][y-2] == chip and gameBoard[x+3][y-3] == chip:
        print("\nJeu terminé", chip, "gagne ! Merci d'avoir joué :)")
        return True

  ### Check upper left to bottom right diagonal spaces
  for x in range(rows - 3):
    for y in range(cols - 3):
      if gameBoard[x][y] == chip and gameBoard[x+1][y+1] == chip and gameBoard[x+2][y+2] == chip and gameBoard[x+3][y+3] == chip:
        print("\nJeu terminé", chip, "gagne ! Merci d'avoir joué :)")
        return True
  return False

def coordinateParser(inputString):
  coordinate = [None] * 2
  if(inputString[0] == "A"):
    coordinate[1] = 0
  elif(inputString[0] == "B"):
    coordinate[1] = 1
  elif(inputString[0] == "C"):
    coordinate[1] = 2
  elif(inputString[0] == "D"):
    coordinate[1] = 3
  elif(inputString[0] == "E"):
    coordinate[1] = 4
  elif(inputString[0] == "F"):
    coordinate[1] = 5
  elif(inputString[0] == "G"):
    coordinate[1] = 6
  else:
    print("Pas valide")
  coordinate[0] = int(inputString[1])
  return coordinate

def isSpaceAvailable(intendedCoordinate):
  if(gameBoard[intendedCoordinate[0]][intendedCoordinate[1]] == '🔴'):
    return False
  elif(gameBoard[intendedCoordinate[0]][intendedCoordinate[1]] == '🔵'):
    return False
  else:
    return True

def gravityChecker(intendedCoordinate):
  ### Calculate space below
  spaceBelow = [None] * 2
  spaceBelow[0] = intendedCoordinate[0] + 1
  spaceBelow[1] = intendedCoordinate[1]
  ### Is the coordinate at ground level
  if(spaceBelow[0] == 6):
    return True
  ### Check if there's a token below
  if(isSpaceAvailable(spaceBelow) == False):
    return True
  return False

leaveLoop = False
turnCounter = 0
while(leaveLoop == False):
  if(turnCounter % 2 == 0):
    printGameBoard()
    while True:
      spacePicked = input("\nChoisissez un espace (exemple D5): ")
      coordinate = coordinateParser(spacePicked)
      try:
        ### Check if the space is available
        if(isSpaceAvailable(coordinate) and gravityChecker(coordinate)):
          modifyArray(coordinate, '🔵')
          break
        else:
          print("Pas un coordonné valide")
      except:
        print("Une erreur s'est produite. Veuillez réessayer.")
    winner = checkForWinner('🔵')
    turnCounter += 1
  ### It's the computers turn
  else:
    while True:
      cpuChoice = [random.choice(possibleLetters), random.randint(0,5)]
      cpuCoordinate = coordinateParser(cpuChoice)
      if(isSpaceAvailable(cpuCoordinate) and gravityChecker(cpuCoordinate)):
        modifyArray(cpuCoordinate, '🔴')
        break
    turnCounter += 1
    winner = checkForWinner('🔴')

  if(winner):
    printGameBoard()
    break
