Recherches et Apprentissages: C++

Que-ce-que c'est?

C++ est un langage de programmation orientée objet qui est très rapide et puissant. Elle est principalement appliquée dans les systèmes d'opérations, applications basées sur l'interface graphique, et systèmes en nuage en raison de sa vitesse et efficacité. C’est un bon idee d'apprendre C++ pour suivre dans un emploi programmation car ces usages sont variés et ca aurait une mauvaise image si tu deviez aller au cours du courbe apprentissage grande pour cette langue quand tu es déjà dans un emploi dont on aurait besoin.

Avantages

C++ est le langage de programmation le plus rapide de tous. En plus, la courbe d'apprentissage est significative avec ce langage, plus intense que Java. C++ est utilisé dans une variété d'applications, y compris les systèmes d'exploitation, les applications avec interface graphique, et les systèmes en nuage, grâce à sa polyvalence et à sa capacité à s'adapter à divers domaines. Contrairement à certains langages de haut niveau, C++ permet aux développeurs de gérer manuellement la mémoire, offrant un contrôle fin sur les ressources et contribuant à la performance

Desavantages

En plus, la courbe d'apprentissage est significative avec ce langage, plus intense que Java. L’autre désavantage majeur du langage C++ est le bibliotech limiter, la grande majorité de ce qui est fait avec C++ est déjà là dans le langage lui-même et non ajouter après. Le développement en C++ peut parfois nécessiter plus de temps en raison de la nécessité de gérer des détails bas niveau, bien que cela puisse également être considéré comme un avantage pour des applications nécessitant une optimisation poussée.

                    return 1;

    //This is a vital function in this program , it can either place or remove a piece,
    void placePiece(string playerPiece, string playerOneSymbol, string playerTwoSymbol, int playerCol, bool undo)
    {
        //This function places a piece by starting from the bottom up in a given column, and checking if there is a free cell, if so it places a piece
        for (int i = NUMROWS - 1; i >= 0; i--)
        {
            if (boardArray[i][playerCol] == "|   " && undo == false)
            {
                //Places the piece at the closest valid coordinate to the bottom row (to emuilate gravity)
                boardArray[i][playerCol] = "| " + playerPiece + " ";

                //Then the function will set the current coordinate virables as the coordinates of the cell where the piece was just placed, dependent on who's turn it is
                if (playerPiece == playerOneSymbol)
                {
                    p1CurrentX = i;
                    p1CurrentY = playerCol;
                }
                else if (playerPiece == playerTwoSymbol)
                {
                    p2CurrentX = i;
                    p2CurrentY = playerCol;
                }
                //This adds 1 to the movesplayed counter variable
                addMovePlayed(1);
                break;
            }

        }
        //This function removes a piece by using the currentX and Y values that were set when a piece was placed and then setting that coordinate to an empty cell. This process is also dependent on the current turn
        if (undo)
        {
            if (playerPiece == playerOneSymbol)
            {
                if (boardArray[getP1CurrentX()][getP1CurrentY()] == "| " + playerPiece + " ")
                {
                    //If the current coordinates do have a piece in them, remove that piece(set it to an emtpy cell)
                    boardArray[getP1CurrentX()][getP1CurrentY()] = "|   ";
                    //Decrease the movesplayed counter by 1
                    addMovePlayed(-1);
                }
            }
            else if (playerPiece == playerTwoSymbol)
            {
                if (boardArray[getP2CurrentX()][getP2CurrentY()] == "| " + playerPiece + " ")
                {
                    //If the current coordinates do have a piece in them, remove that piece(set it to an emtpy cell)
                    boardArray[getP2CurrentX()][getP2CurrentY()] = "|   ";
                    //Decrease the movesplayed counter by 1
                    addMovePlayed(-1);
                }
            }
        }
    }

            {
                cout << "Player Two has WON ";

    //These functions are used in the minimax algorithm to determine the value of a given position
    //This function goes through and checks each column vertically, and adds the amount of each piece to a count variable, then if there are pieces, it adds to the score variable based on that
    int evaluateVertical(string playerOneSymbol, string playerTwoSymbol)
    {
        int score = 0;
        //Go over each cell vertically and add the amount of each players piece to their designated variable
        for (int i = 0; i <= NUMROWS - 4; i++)
        {
            for (int j = 0; j < NUMCOLS; j++)
            {
                int playerOneCount = 0;
                int playerTwoCount = 0;


                for (int k = 0; k < 4; k++)
                {
                    if (boardArray[i + k][j] == "| " + playerOneSymbol + " ")
                    {
                        playerOneCount++;
                    }
                    if (boardArray[i + k][j] == "| " + playerTwoSymbol + " ")
                    {
                        playerTwoCount++;
                    }
                }
                //Add to the score variable based on the amount of pieces squared, and if there is 4 in a row, add a very large value to the score
                if (playerOneCount > 0 && playerTwoCount > 0)
                {
                    continue;
                }
                else if (playerOneCount == 4)
                {
                    score += 1000;
                }
                else if (playerTwoCount == 4)
                {
                    score += -1000;
                }
                else if (playerOneCount > 0)
                {
                    score += playerOneCount * playerOneCount;
                }
                else if (playerTwoCount > 0)
                {
                    score -= playerTwoCount * playerTwoCount;
                }
            }
        }
        //returns the score value
        return score;
    }

                        }
                    }
                    //Add to the score variable based on the amount of pieces squared, and if there is 4 in a row, add a very large value to the score
                    if (playerOneCount > 0 && playerTwoCount > 0)
                    {
                        continue;
                    }
                    else if (playerOneCount == 4)
                    {
                        score += 1000;
                    }
                    else if (playerTwoCount == 4)
                    {
                        score += -1000;
                    }
                    else if (playerOneCount > 0)
                    {
                        score += playerOneCount * playerOneCount;
                    }
                    else if (playerTwoCount > 0)
                    {
                        score -= playerTwoCount * playerTwoCount;
                    }
                }
            }
        }
        //returns the score after all caluclations
        return score;
    }
    //This function, similar to the checkWin function, combines all of the different evaluation functions into one function to return a fully evaluated position
    int evaluatePosition(string playerOneSymbol, string playerTwoSymbol)
    {
        //If the game is over, that means that someone has won, meaning that the score needs to be very large
        if (checkGameOver())
        {
            if (checkVertWin(playerOneSymbol, playerTwoSymbol) == 1)
            {
                return 2000;
            }
            else if (checkVertWin(playerOneSymbol, playerTwoSymbol) == 2)
            {
                return -2000;
            }
            if (checkHorizontWin(playerOneSymbol, playerTwoSymbol) == 1)
            {
                return 2000;
            }
            else if (checkHorizontWin(playerOneSymbol, playerTwoSymbol) == 2)
            {
                return -2000;
            }
            if (checkDiagonalWin(playerOneSymbol, playerTwoSymbol) == 1)
            {
                return 2000;
            }
            else if (checkDiagonalWin(playerOneSymbol, playerTwoSymbol) == 2)
            {
                return -2000;
            }
            return 0;
        }

        //These variables are used to store the evalutaion of each direction, and then return it as a score variable
        int score = 0;

        //These are weight variables, that are copared to the ratio of empty spaces to the total amount of spaces, this is done to change the value of different win types, based on different stages of the game. i.e: early game, mid game, late game
        int numEmpty = getNumEmptySpaces();
        int totalSpaces = 42;
        double emptyRatio = static_cast<double>(numEmpty) / totalSpaces;
        double vertWeight = 1.0;
        double horizWeight = 1.0;
        double diagWeight = 1.0;

        //In the early game, horizontal wins have a higher value, as they are usually pretty easy to build in the early game
        if (emptyRatio >= 0.7)
        {
            vertWeight = 1;
            horizWeight = 1.5;
            diagWeight = 1.25;

        }
        //In the mid game, everything is almost equal, except that vertical scores are worth more since the game needs to get higher and higher in the board
        else if (emptyRatio >= 0.4)
        {
            vertWeight = 1.25;
            horizWeight = 1;
            diagWeight = 1;
        }
        //At the end game everything is more a bit more, since winning is very important when there are limited spots
        else
        {
            vertWeight = 1.25;
            horizWeight = 1.25;
            diagWeight = 1.25;
        }


        //adds these weighted score values to a score variable and returns it
        score += (vertWeight * evaluateVertical(playerOneSymbol, playerTwoSymbol));
        score += (horizWeight * evaluateHorizontal(playerOneSymbol, playerTwoSymbol));
        score += (diagWeight * evaluateDiagonal(playerOneSymbol, playerTwoSymbol));

        return score;

    }
    //This method gets the amount of free spaces, by subtracting the total spaces by the amount of spaces taken by a move
    int getNumEmptySpaces()
    {
        return 42 - getMovesPlayed();
    }

    //This is the mininmax function, the function where the bulk of the work for the ai happens.
    //In this function, it goes down to the depth specified, and then gets the value of each move by placing a piece, evaluating the position, storing that value, then erasing that piece, and then comparing that value to every previous value, to determine the best move
    int minimax(string playerOneSymbol, string playerTwoSymbol, int depth, int alpha, int beta, bool isMaxPlayer)
    {
        //When the depth is 0 or the game is over, return the evaluation of the poisiton
        if (depth == 0 || checkGameOver())
        {
            return evaluatePosition(playerOneSymbol, playerTwoSymbol);
        }
        //If it is player one making moves they want to make the score as high as possilbe
        if (isMaxPlayer)
        {
            int value = INT_MIN;
            //Go through every valid column , and get the biggest value using the "max" operation
            for (int i = 0; i < NUMCOLS; i++)
            {
                if (!colIsFull[i])
                {
                    if (depth == 1)
                    {
                        placePiece(playerOneSymbol, playerOneSymbol, playerTwoSymbol, i, false);
                    }

                    //Here the function calls itself, with paramaters that go one level deeper, with a different player making the moves, because moves are played sequentially
                    //Because of this, the caluclations are done when depth = 1, since below that, it is the evaluation that is given, and not the program going one layer deeper
                    value = max(value, minimax(playerOneSymbol, playerTwoSymbol, depth - 1, alpha, beta, false));

                    //This function was deleting too many pieces when it was at a depth of 2, so this if statement limits that
                    if (depth != 2)
                    {
                        placePiece(playerOneSymbol, playerOneSymbol, playerTwoSymbol, i, true);
                    }

                    //use alpha beta pruning, in other words, remembering the best possible value that it can get, and ruling out "branches" (by breaking the loop) if the program knows that it has a better move no matter what. It knows this by storing the best value for player one as alpha, and for player two as beta
                    if (value > beta)
                    {
                        break;
                    }
                    alpha = max(alpha, value);

                }
            }
            //returns the best value that it found
            return value;
        }
        //If it is player two making moves they want to get the score as low as possible
        else
        {
            int value = INT_MAX;
            //Go through every valid column , and get the smalled value using the "min" operation
            for (int i = 0; i < NUMCOLS; i++)
            {
                if (!colIsFull[i])
                {
                    if (depth == 1)
                    {
                        placePiece(playerTwoSymbol, playerOneSymbol, playerTwoSymbol, i, false);
                    }
                    //Here the function calls itself, with paramaters that go one level deeper, with a different player making the moves, because moves are played sequentially
                    //Because of this, the caluclations are done when depth = 1, since below that, it is the evaluation that is given, and not the program going one layer deeper
                    value = min(value, minimax(playerOneSymbol, playerTwoSymbol, depth - 1, alpha, beta, true));

                    //This function was deleting too many pieces when it was at a depth of 2, so this if statement limits that
                    if (depth != 2)
                    {
                        placePiece(playerTwoSymbol, playerOneSymbol, playerTwoSymbol, i, true);
                    }

                    //use alpha beta pruning, in other words, remembering the best possible value that it can get, and ruling out "branches" (by breaking the loop) if the program knows that it has a better move no matter what. It knows this by storing the best value for player one as alpha, and for player two as beta
                    if (value < alpha)
                    {
                        break;
                    }
                    beta = min(beta, value);

                }
            }
            //returns the best value that it found
            return value;
        }

    }
    //This function find the best move for player one, it is not used directly in the main function, since the ai plays as player two, but this was usefull for testing
    int findBestMove(string playerOneSymbol, string playerTwoSymbol)
    {
        //Initializing variables to store the best values and the current coordinates
        int bestVal = INT_MIN;
        int bestMove;
        int cX = 0;
        int cY = 0;

        //checks and does changes to see if there are any invalid columns before proceeding to find the best move
        for (int i = 0; i < NUMCOLS; i++)
        {
            if (checkTopRowHasPiece(i))
            {
                colIsFull[i] = true;
            }
            else
            {
                colIsFull[i] = false;
            }
        }
        //For every valid column
        for (int i = 0; i < NUMCOLS; i++)
        {
            if (!colIsFull[i])
            {
                //place a piece, then store the current x and y cordinates of that piece, then run the minimax function on this position.
                placePiece(playerOneSymbol, playerOneSymbol, playerTwoSymbol, i, false);
                cX = getP1CurrentX();
                cY = getP1CurrentY();
                //At this point, since there is a depth of 4, the minimax algorithm will initiate and go deep down to the bottom layer, and come back and return the value of that position, while placing moves and checking future moves (4 turns in the future)
                int moveVal = minimax(playerOneSymbol, playerTwoSymbol, 4, INT_MIN, INT_MAX, true);
                //Then, once this evaluation of the poisiton is gotten, the coordinates of the piece that was placed originally are set to be the most recent coorindates, which are then reset
                p1CurrentX = cX;
                p1CurrentY = cY;
                placePiece(playerOneSymbol, playerOneSymbol, playerTwoSymbol, i, true);
                cout << "I: " << i << " VALUE: " << moveVal << "\n";
                //Then like in the minimax function, the best move is determined by comparing the current value to the previous one, in this case we want the largest value
                if (moveVal > bestVal)
                {
                    //Based on this, the best move variable is set to the current column which has the highest value and the bestValue is set to the current value
                    bestMove = i;
                    bestVal = moveVal;
                }
            }
        }
        //The best move is then returned
        return bestMove;
    }
    //This function find the best move for player two, in this case, the AI. The only 2 major differences between this and the previous findBestMove is that there is a depth parameter here, and there are some simple overides to avoid a guarenteed player one win at the begining of the game
    int findBestMoveTwo(string playerOneSymbol, string playerTwoSymbol, int depth)
    {
        //Initializing variables to store the best values and the current coordinates
        int bestVal = INT_MAX;
        int bestMove;
        int cX = 0;
        int cY = 0;
        //checks and does changes to see if there are any invalid columns before proceeding to find the best move
        for (int i = 0; i < NUMCOLS; i++)
        {
            if (checkTopRowHasPiece(i))
            {
                colIsFull[i] = true;
            }
            else
            {
                colIsFull[i] = false;
            }
        }
        //For every valid column
        for (int i = 0; i < NUMCOLS; i++)
        {
            if (!colIsFull[i])
            {
                //place a piece, then store the current x and y cordinates of that piece, then run the minimax function on this position.
                placePiece(playerTwoSymbol, playerOneSymbol, playerTwoSymbol, i, false);
                cX = getP2CurrentX();
                cY = getP2CurrentY();
                //At this point, based on the depth paramater which is set by the player as "difficulty", the minimax algorithm will initiate and go deep down to the bottom layer, and come back and return the value of that position, while placing moves and checking future moves ("depth" turns in the future)
                int moveVal = minimax(playerOneSymbol, playerTwoSymbol, depth, INT_MIN, INT_MAX, false);
                //Then, once this evaluation of the poisiton is gotten, the coordinates of the piece that was placed originally are set to be the most recent coorindates, which are then reset
                p2CurrentX = cX;
                p2CurrentY = cY;
                placePiece(playerTwoSymbol, playerOneSymbol, playerTwoSymbol, i, true);
                //Then like in the minimax function, the best move is determined by comparing the current value to the previous one, in this case we want the lowest value
                if (moveVal < bestVal)
                {
                    //Based on this, the best move variable is set to the current column which has the highest value and the bestValue is set to the current value
                    bestMove = i;
                    bestVal = moveVal;
                }
            }
        }

        //These are some simple overide functions, that check to avoid a situation where player one gets 2 pieces on the bottom, and the ai doesnt block them, making it impossible for player one to win
        //This goes through the bottom row, for every position where it is possible to have 2 pieces in the middle, and 2 free cells on either side, it will overide the decided move to block the guarenteed win
        //This only needs to happen for the bottom row, since after that it is good enough to recognise when the user tries something like that
        if (boardArray[5][1] == "|   " && boardArray[5][2] == "| " + playerOneSymbol + " " && boardArray[5][3] == "| " + playerOneSymbol + " " && boardArray[5][4] == "|   ")
        {
            bestMove = 4;
        }
        if (boardArray[5][2] == "|   " && boardArray[5][3] == "| " + playerOneSymbol + " " && boardArray[5][4] == "| " + playerOneSymbol + " " && boardArray[5][5] == "|   ")
        {
            bestMove = 5;
        }
        if (boardArray[5][3] == "|   " && boardArray[5][4] == "| " + playerOneSymbol + " " && boardArray[5][5] == "| " + playerOneSymbol + " " && boardArray[5][6] == "|   ")
        {
            bestMove = 6;
        }


        //The best move is then returned
        return bestMove;
    }
};
